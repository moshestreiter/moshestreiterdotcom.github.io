Enter valid numbers into the input boxes to run an algorithm that generates a pattern of colors. Make sure to press all the enter buttons for each box.

Hint: Resize the browser window to display various arrays of patterns... Also, open the browser developer console to allow you to resize to narrower widths (which you will want to do when square widths are smaller, just trust me)

Warning: Entering 1 will ask the computer to generate a pattern with 16,777,216 unique colors, and unless your machine has enough RAM (at least 16 gigabytes), the browser renderer will likely not be able to handle this number and the browser window will freeze. If you try a high level on your machine and this happens (not that big a deal if it does), just exit out the window and reopen to start over with a less memory demanding level.
